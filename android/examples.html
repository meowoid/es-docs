Example Applications

Overview

All of the example apps described below (and more) are in the ESLibrary-Examples repository.

Examples

Sensor Manager Basic Example. A simple app that shows you how to sense from all the different sensors using our API.

Sensor Manager Basic Example. An app with a basic UI that allows you to select and sense from a sensor.

Sensor Data Manager Example. An app that shows you how to extend one of our logger types in order to store and transfer the data you sense.

Sensing with Service Subscriptions Example. An app that senses using two different background services.

Sensing with Alarms Example. An app that shows you how to combine alarms and sensing.
