Data Queries

Recent Data

The data manager instance includes:

public List<SensorData> getRecentSensorData(int sensorId, long startTimestamp) throws ESException, IOException;

Where the sensorId is the sensor's integer identifier from com.ubhave.sensormanager.sensors.SensorUtils.java; the query returns any sensor data captured since the startTimestamp is (in millis) until now.
