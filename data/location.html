Location

Definition

This sensor provides the coarse-grained (i.e., cell tower, low accuracy) or fine-grained (i.e., GPS, high accuracy) geographic location of the device.

Data Format

The raw data provides the sensed latitude, longitude, and accuracy.

{
	"sensorType": "Location",
	"senseStartTime":"HH:mm:ss:SSS dd MM yyyy Z z",
	"senseStartTimeMillis":0,
	"userid":"string",
	"deviceid":"string",
	"postSenseSleepMillis":0,
	"isAdaptivelySensed":false,
}

Permissions

<uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION" >
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" >

Usage

ESSensorManager sm = ESSensorManager.getSensorManager(context);
LocationData d = (LocationData) sm.getDataFromSensor(SensorUtils.SENSOR_TYPE_LOCATION);
