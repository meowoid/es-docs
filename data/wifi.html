WiFi

Definition

The WiFi sensor performs a scan of nearby WiFi networks.

Data Format

The scan result is formatted as a list of JSON objects with each network's SSID, BSSID, capabilities, level, and frequency. If there are no results, the scanResult list contains the string "unavailable".

{
	"sensorType": "WiFi",
	"senseStartTime":"HH:mm:ss:SSS dd MM yyyy Z z",
	"senseStartTimeMillis":0,
	"userid":"string",
	"deviceid":"string",
	"postSenseSleepMillis":0,
	"isAdaptivelySensed":false,
	"scanResult":
	[
		{
			"ssid":"string",
			"bssid":"string",
			"capabilities":"string",
			"level":int,
			"frequency":int
		}
	]
}
