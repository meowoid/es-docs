Overview

The ubiquity of sensor-rich and computationally powerful smartphones makes them an ideal platform for health and wellness applications. To facilitate the development of social sensing and data collection applications, we are developing a set of open-source smartphone libraries to collect, store and transfer, and query sensor data. All of our code, including these documentation pages, can be found on our Github Page.

Sensing API

The Sensor Manager (Android Code, Android Docs) provides a uniform and easily configurable access to sensor data, supporting both one-off and continuous sensing scenarios.

JSON Formatting

We use an open format to represent data from each sensor, using JSON. In the data section of these docs, you can find descriptions and example data from all of the sensors that our library can currently sense from; the Data Manager library includes methods to format data.

Data Storage and Transmission

To encapsulate the ideas of formatting, storing, querying, and transmitting data, we built a Data Manager (Android Code, Android Docs) that acts as a data sink for sensor-enhanced applications. The manager directly supports formatting sensor data and can also store and transmit any data that is identified with an appropriate tag. Since applications may collect large amounts of data, the manager can also be configured to transmit data asynchronously.
